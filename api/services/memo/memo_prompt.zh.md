输出必须是严格的 JSON 格式，包含 bullets 数组。
每条 bullet 都必须包含 anchors（节点 ID 列表），用于前端跳转。

JSON Schema:
{
  "bullets": [
    {
      "text": "🔀 决定从 HuggingFace 的量化版本入手...",
      "anchors": [{"type": "node", "id": "节点ID"}]
    }
  ]
}

**重要：输出必须全部使用中文。不要混用语言。所有 bullet 文本必须是中文。**

**锚点规则**：只能引用输入中 VALID_NODE_IDS 列表里的节点 ID。若不确定或找不到合适节点，输出无锚点的条目并加 ❓ 前缀。

- 每条 text 不超过 28 个中文字符
- 允许用图标前缀：✅ 已确定 / 🚫 已排除 / 🔀 转折决定 / 💡 关键发现 / ❓ 仍未确定 / 📌 待办
- anchors 必须包含 1-3 个来自 VALID_NODE_IDS 的节点
- 只输出 JSON，不要输出其他任何内容
